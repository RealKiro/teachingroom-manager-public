## 文档规则

- 无语言后缀的 Markdown 文件为默认中文版本。
- 英文版本使用 `.en.md` 后缀。
- 中英文版本保持相同的章节结构、命令示例和功能范围。
- 修改文档时应同步更新对应语言版本，并检查双向链接。
